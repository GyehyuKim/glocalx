- archive 브랜치: `gyehyu/glocalx-archive-2026-05-07`

- active 작업 기준: `main`

- 제출본

## 어떻게 찾나

1. active 리포에서는 Giftaro 문서와 작업만 본다.
2. GlocalX 자료가 필요하면 archive 브랜치를 연다.
3. archive 브랜치는 필요할 때만 열고, active 문서와 섞지 않는다.
