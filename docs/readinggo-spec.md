## 1. 서비스 개요

| 항목 | 내용 |
|---|---|
| 서비스명 | 리딩고 (Reading GO) |
| 마스코트 | 참새 리리 (Riri) |
| 레퍼런스 | 듀오링고 게이미피케이션 메커니즘을 독서 여정에 최적화 |
| 핵심 문제 | 독서를 시작하려는 의도는 있지만, 첫 페이지를 여는 마찰력이 너무 크다 |
| 해결 방식 | 연속적인 소규모 보상 + 시각적 성장 피드백 + 소셜 피어 프레셔 |

## 1.1 제품 약속 (Promises)

| 사용자가 얻는 것 | 제품이 책임지는 것 |
|---|---|
| 하루 1페이지 이상 읽는 습관 형성 | 듀오링고 수준의 지속성 엔진 |
| 읽은 책의 핵심을 손에 남긴다 | "오늘의 문장" 누적 → 책 한 권의 엑기스 → export |
| 혼자가 아니라 같이 읽는다 | 같은 책 피드, 박수, 공동 자산, 클럽 |

타겟: **"읽고 싶은데 이어나가지 못하는 사람"**. 안 읽는 사람을 끌어오는 제품 아님.

### 1.2 왜 책 / 왜 페이지인가

- **모두가 하고 싶어하는 행동** — "더 읽고 싶다"는 보편적 욕구. 끌어올 동기가 이미 있음.
- **최소한의 정량화가 가능한 유일한 카테고리** — 페이지는 객관적 단위. 운동(강도/시간/종목 제각각), 식사(질/양 측정 불가), 음주(부정적 방향)와 달리 책은 "1페이지"가 명확한 진척 단위.

## 2. 핵심 기능

### 2-1. 동적 스테이지 (Dynamic Stage)

정해진 목차가 아닌 **사용자의 기록 횟수**에 따라 실시간으로 생성되는 로드맵.

- 기록 1회 = 로드맵 1칸 전진
- 목표 페이지/완독일 설정 없음 — **1페이지만 읽어도 오늘은 성공**
- 칸 수는 책의 전체 페이지 수와 무관하게 누적 기록에 비례

### 2-2. 성장형 비주얼 피드백 — 집 진화 시스템

읽은 페이지 비율(진행률 %)에 따라 마스코트 리리의 집이 5단계로 진화한다.

> 진행률 = (오늘 읽은 마지막 페이지 / 전체 페이지 수) × 100

| 진행률 | 단계명 | 비주얼 상세 |
|---|---|---|
| 0 ~ 20% | 기초 공사 | 나뭇가지 몇 개가 놓인 소박한 기초 공사 단계. 리리가 땅바닥에서 독서 시작 |
| 21 ~ 50% | 오두막 (Hut) | 튼튼한 벽체와 입구가 생긴 나무 오두막. 리리가 비를 피할 수 있게 됨 |
| 51 ~ 80% | 예쁜 집 (House) | 지붕이 덮이고 아기자기한 창문이 달린 안정적인 집 형태 |
| 81 ~ 99% | 화려한 저택 (Mansion) | 정원과 야간 조명이 추가된 럭셔리한 대저택. 성공적인 완독 임박 암시 |
| 100% | 리딩 팰리스 (Palace) | 완독 축하 파티가 열리는 궁전. 화려한 연출 + '완독 배지' 수여 |

### 2-3. 리딩 빌리지 (Reading Village)

- 친구들의 집 목록을 마을 형태로 시각화
- 집 불빛 ON/OFF = 오늘 독서 여부를 한눈에 파악
- **'모이(Worm) 보내기'**: 아직 읽지 않은 친구에게 독서 독려 알림 전송
- 피어 프레셔(Peer Pressure)를 긍정적 동기 부여로 전환

## 3. 유저 저니 (User Journey)

```
[책 등록] → [데일리 리딩] → [보상 획득] → [소셜 인터랙션]
```

### Step 1. 책 등록

- ISBN 검색으로 책 선택
- 알라딘/네이버 책 API로 표지·저자·페이지 수 자동 확인
- 목차 분할: 알라딘 프리미엄 가능 시 자동 / fallback: 데모 20권 사전 입력 / 사용자 수동 입력
- **MVP 데이터**: 베스트셀러 10권 + 스테디셀러 10권 = 총 20권 사전 입력

### Step 2. 데일리 리딩

매일 한 페이지라도 읽고 기록한다.

| Step | 입력 | 검증 |
|---|---|---|
| 1. 진입 | 미션 시작 버튼 탭 | — |
| 2. 현재 위치 입력 | 오늘 읽은 마지막 페이지 (단일 입력) | 이전 기록 ≤ 입력 ≤ 책 총 페이지 |
| 3. **오늘의 문장 수집 (강제)** | 기억하고 싶은 한 문장. 최소 10자, 최대 500자 | 빈 입력 차단. 자동 임시 저장 |
| 4. 세리머니 | XP +50, 스트릭 +1, 참새 hop 애니메이션 | — |

> **왜 오늘의 문장이 강제인가**: "읽었다"의 최소 검증 조건이며, 누적되면 책의 엑기스가 되어 사용자에게 진짜 자산으로 남는다.

### Step 3. 보상 획득

기록 완료 즉시 아래 세 가지 보상이 동시에 지급된다.

| 보상 | 내용 |
|---|---|
| 로드맵 전진 | 동적 스테이지 1칸 전진 |
| XP 획득 | 경험치 누적 → 레벨업 |
| 스트릭(🔥) 유지 | 연속 기록일 수 카운트 유지 |

### Step 4. 소셜 인터랙션

- 리딩 빌리지에서 친구의 집 불빛 확인
- 오늘 기록 없는 친구에게 '모이(Worm)' 전송
- 같은 책 피드에서 다른 독자의 문장에 박수

## 4. 화면 스펙

### 4.1 홈 — The Path

**메타포**: 책의 목차가 듀오링고 학습 트리의 스테이지.

| 요소 | 동작 |
|---|---|
| 상단 바 | 🔥 Streak (연속일) / 💎 XP / 🪶 방패 보유 수 |
| The Path | 챕터별 스테이지를 구불구불한 길에 배치. 완료/현재/잠금 상태 시각화 |
| 캐릭터 | 참새 리리. 현재 진행 챕터에 위치. 미션 완료 시 다음 노드로 hop |
| CTA | "오늘의 미션 시작" 버튼 |
| 하단 탭 | 홈 / 피드 / 랭킹 / 내 서재 |

상태 표시 규칙:
- 챕터 완료: 색 채워짐
- 현재 챕터: 펄스 애니메이션 + 챕터 내 페이지 진척 미니 프로그레스 바 (e.g. "23/40p")
- 미열림 챕터: 회색 + 잠금 아이콘

### 4.2 소셜 피드 — 공동 독서

| 섹션 | 콘텐츠 |
|---|---|
| 같은 책 피드 | 내가 읽는 책을 같이 읽는 다른 사람의 "오늘의 문장". 표지/페이지/문장/박수 |
| 친구 피드 | 팔로우한 친구의 미션 완료 카드 |
| 박수 | 1-탭 반응. 알림 X (마찰 최소화). 누적 카운트만 |

- 댓글 X (MVP). 박수와 스티커로만 응답 → 인지 부담 최소
- 비공개 모드: 문장을 본인만 보기 가능. 스트릭/공동 자산 기여에는 영향 없음

### 4.3 랭킹 / 클럽

| 단위 | 산정 |
|---|---|
| 주간 개인 랭킹 | XP 합산 |
| 클럽 (책 단위) | 같은 책 읽는 사람들의 묶음. 자동 가입 옵션 |
| 공동 자산 | 클럽 단위로 "도서관 짓기" / "세계수 키우기" — 미션 완료마다 1 brick / 1 leaf |

### 4.4 내 서재 / 문장 모음

| 화면 | 내용 |
|---|---|
| 책 목록 | 진행 중 / 완독 |
| 책 상세 | 누적된 "오늘의 문장" 타임라인 |
| **Export** | Markdown (책 제목 + 문장 목록 + 수집일자) |

## 5. 시스템 로직

### 5.1 스트릭

| 항목 | 규칙 |
|---|---|
| 갱신 시점 | 매일 23:59 (사용자 로컬 타임존) |
| 갱신 조건 | "오늘의 문장" 1개 이상 제출 |
| 미참여 시 | 불꽃 꺼짐. 방패로 완화 가능 |

### 5.2 회복탄력성 (What-the-hell 방어)

| 장치 | 동작 |
|---|---|
| 🪶 방패 (Shield) | 자동 적용. 1일 미참여 = 방패 1개 소모하고 스트릭 유지. 7일마다 1개 무료 충전 |
| ⏸️ 휴식권 (Pause) | 사용자가 사전에 1-7일 휴식 선언. 스트릭 동결. 월 1회 무료 |
| 복귀 메시지 | 결석 후 복귀 시 죄책감 유발 X. "다시 시작" 톤만 |

### 5.3 XP / 배지

| 행동 | XP |
|---|---|
| 일일 미션 완료 | +50 |
| 챕터 완료 | +100 |
| 책 완독 | +500 |
| 친구에게 박수 받음 | +5 (일일 최대 +20 cap) |
| 7일 연속 | 배지 + XP +200 |
| 30일 연속 | 배지 + XP +1000 |

## 6. 기대 효과 및 리텐션 전략

### 손실 회피 심리 (Loss Aversion)
공들여 지은 '저택'을 무너뜨리지 않기 위해 스트릭을 유지하려는 강한 동기 부여.

### 가시적 성취 (Tangible Progress)
추상적인 독서량을 **집의 크기**와 **길의 길이**라는 물리적 시각 정보로 치환 → 진행감 체감.

### 공동체 의식 (Sense of Community)
혼자가 아닌 '마을'의 일원으로서 함께 성장한다는 유대감 형성.

## 9. MVP 범위 (Cut Line)

**포함**:
- The Path, 일일 미션, "오늘의 문장" 강제 수집
- 스트릭 + 방패 + 휴식권
- 집 진화 시각화 (5단계)
- 같은 책 피드, 박수
- 주간 개인 랭킹 + 클럽 공동 자산 (1개 테마)
- 내 서재 + Export (Markdown)
- 책 20권 사전 입력

**제외 (Post-MVP)**:
- 댓글
- 알라딘 프리미엄 API 연동
- 결제 / 방패 추가 구매
- 이미지 카드 export
- 푸시 알림 (네이티브)

## 10. 데모 시나리오 (#58용)

3분 시연 흐름:

1. **(0:00)** 홈 진입 — The Path에 사피엔스가 30% 진행 상태 (오두막 단계) 표시
2. **(0:30)** 미션 시작 → 페이지 입력 → 오늘의 문장 입력 → 세리머니
3. **(1:00)** 피드 진입 — 다른 사람의 문장 1개에 박수
4. **(1:30)** 클럽 → 도서관이 자라나는 모습 보여줌
5. **(2:00)** 내 서재 → 누적 문장 → Export 1회 클릭으로 markdown 다운
6. **(2:30)** "안 읽은 다음날" 시뮬레이션 → 방패 자동 사용 → 스트릭 유지 메시지
