# ReadingGo 도서 데이터 수집 프로세스

## 개요
ReadingGo MVP를 위한 548권 한국/세계문학 데이터셋. 메타데이터(ISBN, 저자, 페이지수, 커버 이미지)와 목차 정보를 포함.

## 수집 단계

### 단계 1: 초기 100권 수집 (Opus, 2026-05)
- **범위**: b001-b100
- **출처**: 알라딘 API (Aladin CDN cover URLs)
- **방식**: Python urllib + 정규표현식으로 og:image 메타태그 추출
- **검증**: HTTP HEAD 요청으로 모든 커버 URL 확인 (10/10 working)

### 단계 2: 민음사 세계문학전집 10권 추가 (Haiku, 2026-05)
- **범위**: b101-b110
- **출처**: 알라딘 상품페이지 크롤링
- **방식**: 개별 ISBN 검색 → ItemId 추출 → 상품페이지에서 메타데이터 수집
- **검증**: 모든 ISBN 유효성 확인, 커버 URL 동작 확인

### 단계 3: 민음사 세계문학전집 확대 수집 (Gemini, 2026-05)
- **범위**: b111-b548 (438권)

- **방식**: 대량 자동 수집

## 데이터 형식

### books.tsv (최종 메타데이터)
```
book_id | isbn | title | author | publisher | total_pages | cover_url
b001    | 9788934972464 | 사피엔스 | 유발 하라리 | 김영사 | 648 | https://...
```
- **행**: 548권 + 헤더
- **인코딩**: UTF-8
- **구분자**: 탭 (제목의 쉼표 포함으로 인한 CSV 파싱 문제 회피)

### books_toc.csv (목차 데이터)
```
book_id | isbn | title | author | publisher | total_pages | chapter_order | chapter_title | start_page | end_page
```
- **행**: 548권 + 각 책의 장/목차
- **현황**: 대부분 "본문" 단일 항목 (일부 상세 목차 포함)
- **목적**: 독서 진도 추적, 챕터별 학습 경험

### covers-preview.html
- **용도**: 548권 커버 시각화 및 프론트엔드 데모
- **출력**: 책 ID, 제목, 저자, Aladin CDN 이미지

## 주의사항

1. **제목에 쉼표 포함**: 예) "변신, 시골의사" → CSV 파싱 문제 → TSV 형식 사용
2. **커버 URL**: Aladin CDN (image.aladin.co.kr) 기반 → 외부 의존성
3. **목차 정보**: 단순 "본문" 표시인 책이 다수 (추후 개선 가능)
