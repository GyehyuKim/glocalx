# Premortem: GlocalX

> "2026년 6월 20일 Demo Day. GlocalX는 실패했다. 무엇이 잘못되었는가?"

## Impact 기준

- **Critical**: 프로젝트 자체가 멈춤. 우회 방법 없음.
- **High**: 심각한 타격이지만 피벗/우회 가능.
- **Medium**: 아프지만 학기 내 치명적이지 않음.

## Marketing / Brand (승원)

| # | 실패 시나리오 | Impact |대응 방안 |
|---|-------------|--------|----------|
| 3 | 구글이 GBP 다국어 자동번역을 네이티브로 출시하거나, 네이버·인스타 대행사가 GBP 시장에 진입 | Medium |구글이 단순 번역을 제공해도, GlocalX의 핵심은 번역이 아니라 지속적 게시 자동화 + 한국 음식 카테고리 특화 콘텐츠다. "국밥"을 "Rice Soup"로 번역하는 게 아니라 "Hearty Busan-style hangover soup, a local staple since the 1950s"로 맥락화하는 것은 범용 번역이 대체할 수 없다. 경쟁 진입 시 이 콘텐츠 품질 격차를 핵심 방어선으로 삼는다. 또한 초기에 점주와의 관계 자산(리뷰 히스토리, 프로필 최적화 누적치)을 빠르게 쌓아두면 후발 진입자가 끊기 어렵다. |
| 4 | GBP 노출은 늘었는데 외국인이 실제로 방문하지 않는다. 길찾기 클릭이 발생하지 않거나, 클릭 이후 이탈한다. | High |실패 원인을 두 단계로 분리해서 진단한다. 첫째, 노출 → 클릭 실패면 콘텐츠 문제다. 외국인이 실제로 반응하는 키워드와 사진 포맷을 PoC 기간 내 A/B 테스트로 빠르게 검증한다. 리서치에 따르면 외국인은 "현지인 리뷰"와 "최신 사진"을 기반으로 방문을 결정한다 — 사진 퀄리티와 업로드 주기가 핵심 변수다. 둘째, 클릭 → 방문 실패면 정보 정확도 문제다. 영업시간·위치 불일치가 이탈을 만든다. 자동 동기화 정확도를 최우선으로 관리한다. 최종 안전망은 북극성 지표를 '노출 수'가 아닌 '길찾기 클릭'으로 고정한 것 자체다 — 노출이 늘어도 클릭이 없으면 실패로 인식하고 즉시 콘텐츠 전략을 수정한다.|
