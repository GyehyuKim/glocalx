# 트렌드 패치노트 (Trend Patchnote)

> 3040 마케터를 위한 트렌드 GAP 진단 + 주간 큐레이션 AI 에이전트

## 컨셉

인스타 알고리즘은 사람마다 다른 트렌드를 보여줍니다. 시니어 마케터들이 주니어들의 밈을 놓치는 이유죠. **트렌드 패치노트**는 10문항 진단으로 당신이 새고 있는 영역을 찾아내고, 매주 그 영역만 골라서 보내드립니다.

## MVP 데모

`index.html` 파일을 브라우저에서 열면 동작합니다.

- 진단지 (5문항 인터랙티브)
- 로딩 → 결과 분석 화면
- 영포티 유형 5단계 + 약한 영역 게이지 분해
- 주간 패치노트 샘플 + 구독 CTA

## 타겟 & 차별화

| 구분 | 우리 | 캐릿 / 일반 뉴스레터 |
|---|---|---|
| 대상 | 3040 마케터 | 전 연령 |
| 방식 | 약점 진단 → 맞춤 큐레이션 | 모두에게 같은 푸시 |
| 가치 | "당신이 놓친 트렌드" | "이번 주 트렌드" |
