# 찍먹

> 찍어보면 알아. 사기 전에 답해주는 앱.

## 컨셉

2030 당뇨전단계·인슐린저항성 직장인을 위한 식품 안전성 진단 앱. 편의점 음료 사진 한 장이면 식후 혈당 반응을 1초 만에 예측해드려요.

## 슬로건

**이거 먹어도 돼? — 찍어보면 알아.**

## 타겟 & 차별화

- **타겟**: 2030 당뇨전단계·인슐린저항성·대사증후군 직장인 (한국 ~1500만 명 추정)
- **차별화**:
  - Yuka/일반 성분 분석 → "이 제품의 성분 점수"만 알려줌
  - 글루코핏 등 CGM 연동 앱 → 월 10만원+ 장비 필요
  - 찍먹 → **CGM 없이, 사기 전에, 당신 프로파일 기준으로** 답

## 핵심 시연

같은 음료를 두 번 스캔:

| | 일반 기준 | 회색지대 유형 (당신) |
|---|---|---|
| 코카콜라 250ml | 12점 / +130mg/dL | **9점 / +175mg/dL** |
| → 35% 더 민감 |

## MVP 데모 흐름

1. 음료 3종 중 하나 탭 → 즉시 스캔
2. 일반 기준 결과 표시 → **"1분 온보딩 진단" CTA**
3. 온보딩 = 가입 후 1회만 (7문항)
4. 프로파일 완성 → 같은 음료 다시 스캔
5. **개인화 점수 + "일반 vs 당신" 비교** ← 차별화 wow 모먼트
